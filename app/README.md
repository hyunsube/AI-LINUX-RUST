# app


## Installation

Install the shared libraray. 
You will get 
- /usr/local/lib/grpc.so
- /usr/local/include/grpc.h

```bash
$ cd app/build
$ cmake ..
$ make
$ sudo make install
```

Compile via the CMakefile 
```bash

$ cd ~/app/build
$ cmake ..

# you can check myapp
$ ./myapp
```
